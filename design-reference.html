Animations

.reveal

Fade-up on scroll (opacity 0→1, translateY 30→0). Applied via IntersectionObserver in delayed.js.

.reveal-children

Staggered child animation — each child fades in with 0.1s delay increment (up to 0.7s for 8th+ child).

.reveal-left / .reveal-right

Slide-in from sides (translateX ±40px → 0).

Hero entrance

Staggered fade-up of h1, p, and button elements with 0.15s delay per item. Managed in hero.js.

prefers-reduced-motion

All animations are disabled when the user has enabled reduced motion. Handled in both hero.js and delayed.js.

Design Rules

  1. --electric-cyan is the primary action color — reserve for CTAs, active states, key data, links
  2. --plasma-violet is secondary — use for gradients and accents alongside cyan
  3. Never use Orbitron for body text or below 20px
  4. Never use Space Grotesk for body/paragraph text (use Inter)
  5. Dark-first: body background is always --carbon-950
  6. Use --status-error for errors, never cyan or violet
  7. Glass morphism: always pair backdrop-filter: blur() with --glass-border
  8. Max 2 accent colors per view (cyan + violet)
  9. Respect prefers-reduced-motion — all animations must be skippable
